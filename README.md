# 电影售票管理系统 java1053

## 项目概述


## 技术栈

- **前端：** Vue
- **后端：** Java、SpringBoot、MyBatis
- **开发环境：** IDEA、JDK1.8、Maven、Node.js
- **数据库：** MySQL

## 系统架构

本系统采用前后端分离的设计模式。

- **前台用户端：** 负责用户注册、登录、购票等操作。
- **后台管理端：** 负责影院信息管理、用户管理等操作。
